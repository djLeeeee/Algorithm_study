## Heap

> heapq 라이브러리 사용하기
>
> ```python
> import heapq
> 
> heapq.heapify(my_list) # my_list를 힙 구조로 변경
> heapq.heappush(my_list, my_num) # 힙 my_list에 원소 my_num 추가
> x = heapq.heappop(my_list) # 힙 my_list에서 최솟값을 뽑아 x에 저장
> ```
